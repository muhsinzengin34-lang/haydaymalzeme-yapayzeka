===========================================
SEÇENEK 1: GitHub Desktop (ÖNERİLEN - KOLAY)
===========================================

1. GitHub Desktop indir: https://desktop.github.com
2. Kur ve GitHub hesabınla giriş yap
3. File > Add Local Repository
4. Bu klasörü seç: c:\Users\EV\Desktop\deneme\haydaydestek-main\haydaydestek-main
5. "Publish repository" butonuna tıkla
6. Repository adı: haydaymalzeme-yapayzeka
7. Public veya Private seç
8. "Publish" butonuna tıkla

✅ Tamamlandı!

===========================================
SEÇENEK 2: Git Komut Satırı
===========================================

1. Git indir: https://git-scm.com/download/win
2. Kur ve bilgisayarı yeniden başlat
3. Bu klasörde terminal aç
4. Şu komutları çalıştır:

git init
git add .
git commit -m "İlk commit: Hay Day Destek Chatbot sistemi"
git branch -M main
git remote add origin https://github.com/KULLANICI_ADINIZ/haydaymalzeme-yapayzeka.git
git push -u origin main

SONRAKI ADIMLAR (Yükleme sonrası)
===========================================

1. .env dosyası oluştur:
   copy .env.example .env

2. .env dosyasını düzenle ve API anahtarlarını ekle:
   OPENAI_API_KEY=your_key_here

3. Bağımlılıkları yükle:
   npm install

4. Sistemi başlat:
   npm run dev
